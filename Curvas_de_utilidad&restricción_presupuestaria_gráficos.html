Curvas de Utilidad¶

Las curvas de indiferencia representan todas las combinaciones de bienes (𝑥,y) que proporcionan el mismo nivel de satisfacción o utilidad (U) para el consumidor. Dado que estamos trabajando con una función de utilidad tipo Cobb-Douglas, podemos derivar la curva de indiferencia a partir de la ecuación de utilidad. Curva de indiferencia con Cobb-Douglas La función de utilidad Cobb-Douglas es:

𝑈(𝑥,𝑦)=𝑥^𝛼⋅y^B=𝑥^𝛼⋅𝑦^(1−𝛼)

Para una curva de indiferencia, fijamos un nivel de utilidad y despejamos una de las variables en función de la otra

# Parámetros
α = 0.5  # Le damos la misma importancia a los dos bienes.
β = 1 - α  # El parámetro de la función Cobb-Douglas puede ser denotado por β.

U0_values = [100, 200, 300, 400, 500, 600, 700, 800, 900, 1000]  # Diferentes niveles de utilidad
x = np.linspace(0.1, 10, 100)  # Valores de x para graficar

# Graficar curvas de indiferencia para diferentes niveles de U_0
plt.figure(figsize=(8, 6))
for U_0 in U0_values:
    y = (U_0 / x**α)**(1 / (1 - α))
    plt.plot(x, y, label=f'U_0 = {U_0}')

plt.xlabel('Bien X')
plt.ylabel('Bien Y')
plt.title('Curvas de Indiferencia - Cobb-Douglas')
plt.legend()
plt.grid(True)
plt.show()

Explicación económica¶

Las curvas de indiferencia muestran todas las combinaciones de (x,y) que proporcionan el mismo nivel de utilidad.

A medida que 𝑈0 aumenta, las curvas se desplazan hacia la parte superior derecha, lo que indica un mayor nivel de satisfacción con distintas combinaciones de cantidades de (x, y).

Cuanto más alejados están los puntos de las curvas de indiferencia de los ejes, mayor es la utilidad que el consumidor está obteniendo, ya que está consumiendo más de ambos bienes.

Este comportamiento es típico en la teoría económica cuando se asume que ambos bienes son complementarios en alguna medida y que el consumidor busca una combinación de ambos bienes para maximizar su satisfacción o utilidad.

¿Qué es el Punto de Equilibrio del Consumidor?¶

Es la combinación óptima de dos bienes que un consumidor elige para maximizar su utilidad, dado su ingreso y los precios de esos bienes.

👉 En otras palabras:

El punto donde se alcanza la curva de indiferencia más alta posible, sin pasarse del presupuesto.

¿Cómo se encuentra matemáticamente?¶

🎯 Punto de Equilibrio del Consumidor¶

Es la combinación óptima de dos bienes que un consumidor elige para maximizar su utilidad, dado su ingreso y los precios de esos bienes.

El punto donde se alcanza la curva de indiferencia más alta posible, sin pasarse del presupuesto**.

✍️ ¿Cómo se encuentra matemáticamente?¶

1. Datos que necesitas:¶

  • Una función de utilidad: Ej. Cobb-Douglas:

[ U(x, y) = x^a . y^b ]

-- Una restricción presupuestaria:
( P_x . x + P_y .y = I )

📐 Método 1: Condición de tangencia¶

La condición para el equilibrio es:¶

[ \frac{MU_x}{MU_y} = \frac{P_x}{P_y} ]

donde:

  • ( MU_x ) y ( MU_y ) son las utilidades marginales de cada bien.
  • ( \frac{MU_x}{MU_y} ) es la Tasa Marginal de Sustitución (TMS).
  • ( \frac{P_x}{P_y} ) es la tasa de intercambio del mercado.

📌 Paso a paso con Cobb-Douglas

Supongamos:
- \( U(x, y) = x^{0.5} y^{0.5} \)
- \( P_x = 1 \), \( P_y = 1 \), \( I = 10 \)

1. Calculamos las derivadas marginales:

\[
MU_x = \frac{\partial U}{\partial x} = 0.5 x^{-0.5} y^{0.5} \\
MU_y = \frac{\partial U}{\partial y} = 0.5 x^{0.5} y^{-0.5}
\]

2. TMS:

\[
\frac{MU_x}{MU_y} = \frac{y}{x}
\]

3. Igualamos a la relación de precios:

\[
\frac{y}{x} = \frac{1}{1} \Rightarrow y = x
\]

4. Sustituimos en la restricción presupuestaria:

\[
x + y = 10 \Rightarrow x + x = 10 \Rightarrow 2x = 10 \Rightarrow x = 5
\Rightarrow y = 5
\]

✅ Solución óptima:  
\( x = 5 \), \( y = 5 \)

## 🧠 Método 2: Lagrangiano

Planteamos:

\[
\mathcal{L}(x, y, \lambda) = U(x, y) + \lambda (I - P_x x - P_y y)
\]

Derivamos:

\[
\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial U}{\partial x} - \lambda P_x = 0 \\
\frac{\partial \mathcal{L}}{\partial y} = \frac{\partial U}{\partial y} - \lambda P_y = 0 \\
\frac{\partial \mathcal{L}}{\partial \lambda} = I - P_x x - P_y y = 0
\]

Resolviendo este sistema obtienes el punto de equilibrio.
